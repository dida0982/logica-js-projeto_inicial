# ⚠️ Pré-requisitos para o Curso de Lógica de Programação

## 📖 Antes de começar

Antes de iniciar este curso, é importante entender que, apesar de ser voltado para iniciantes em programação, ele exige **conhecimentos básicos de informática**.

➡️ O foco aqui é programação, não o uso básico do computador.

---

## 💻 Ambiente de Desenvolvimento

Durante o curso, serão utilizadas as seguintes tecnologias:

- JavaScript
- HTML
- CSS
- VSCode (IDE principal)

Por isso, é essencial ter familiaridade com o uso do computador e do sistema operacional.

---

## 📂 Conhecimentos Necessários

### 📁 Pastas e Arquivos
Você deve saber:
- O que são arquivos e pastas
- Como organizar arquivos
- Como navegar pelo sistema

---

### 🗂️ Diretórios
É importante saber:
- Criar pastas (diretórios)
- Criar subpastas
- Organizar seus projetos

---

### 🔍 Navegação no Sistema
Você precisa conseguir:
- Localizar arquivos no computador
- Acessar diretórios específicos
- Não se perder na estrutura de pastas

---

### 📦 Arquivos .zip
Você deve saber:
- Compactar arquivos
- Descompactar arquivos (.zip)

➡️ Isso será necessário para baixar e organizar materiais.

---

### ⚙️ Instalação de Programas
Você precisa saber:
- Instalar programas
- Desinstalar programas

➡️ Exemplo: instalação do VSCode

---

### 🖥️ Uso de Programas
Você deve ser capaz de:
- Abrir programas instalados
- Navegar entre janelas
- Minimizar e fechar aplicações
- Utilizar menus básicos

➡️ Isso será essencial para usar o VSCode corretamente.

---

## ⚠️ Importante

Este curso **não aborda informática básica**.

Se você ainda não domina esses conceitos, é altamente recomendado aprender antes de continuar.

---

## 🎯 Objetivo

Com esses pré-requisitos atendidos, você poderá:

- Focar totalmente na lógica de programação
- Aproveitar melhor o conteúdo
- Evoluir mais rápido

---

## 🚀 Conclusão

Ter uma base mínima em informática não é opcional — é necessário.

➡️ Isso garante que você consiga acompanhar o curso sem dificuldades técnicas.
