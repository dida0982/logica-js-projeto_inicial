📌 About the Project | Sobre o Projeto

🇺🇸
This is a simple and interactive number guessing game built with JavaScript, HTML, and CSS.
The goal is to guess the secret number randomly generated by the system.

🇧🇷
Este é um jogo simples e interativo de adivinhação desenvolvido com JavaScript, HTML e CSS.
O objetivo é descobrir o número secreto gerado aleatoriamente pelo sistema.

🚀 Features | Funcionalidades
🎲 Random number generation (1 to 10)
🔁 Unlimited attempts
📉 Hints after each guess (higher or lower)
📊 Attempt counter
💬 Interactive alerts and prompts
🧠 Concepts Practiced | Conceitos Praticados
Variables and data types
Conditional logic (if / else)
Loops (while)
Random numbers (Math.random)
User input (prompt)
Alerts and logs
Ternary operator
🖥️ Technologies | Tecnologias
JavaScript
HTML
CSS
🎮 How to Play | Como Jogar

🇺🇸

Open the game in your browser
Enter a number between 1 and 10
Follow the hints
Try until you guess the correct number

🇧🇷

Abra o jogo no navegador
Digite um número entre 1 e 10
Siga as dicas do sistema
Continue tentando até acertar
