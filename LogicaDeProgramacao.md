# 🧠 Lógica de Programação: Fundamentos para Resolver Problemas

## 📖 Sobre este módulo

Este módulo faz parte do guia **Programação do Zero** e tem como objetivo ensinar a base mais importante da programação:

➡️ **Pensar como um programador**

Antes de aprender qualquer linguagem, é essencial entender como resolver problemas de forma estruturada.

---

## 💡 O que é Lógica de Programação?

Lógica de programação é a capacidade de:

- Entender um problema
- Organizar uma solução
- Definir uma sequência de passos

➡️ Em outras palavras: transformar problemas em soluções executáveis.

---

## 📌 O que você vai aprender

- Pensamento lógico
- Criação de algoritmos
- Estruturação de soluções
- Base para qualquer linguagem de programação

---

## 🧩 Algoritmos

Um algoritmo é um **passo a passo** para resolver um problema.

### Exemplo simples:
Acordar
Levantar da cama
Escovar os dentes

➡️ Tudo na programação começa com algoritmos.

---

## 🔁 Estruturas Básicas

### 🔹 Sequência
Execução de passos em ordem.

---

### 🔹 Condição (Decisão)
Permite tomar decisões.

Exemplo:
SE estiver chovendo
levar guarda-chuva
SENÃO
sair normalmente

---

### 🔹 Repetição (Laços)
Executa uma ação várias vezes.

Exemplo:
ENQUANTO tiver tarefas
continuar estudando

---

## 🔢 Variáveis

Variáveis são espaços para armazenar informações.

Exemplo:
idade = 20
nome = "Guilherme"

---

## 🧠 Como pensar como programador

Para resolver qualquer problema:

1. Entenda o problema
2. Divida em partes menores
3. Crie um passo a passo
4. Teste mentalmente a solução

➡️ Esse processo é mais importante que qualquer linguagem.

---

## ⚠️ Erro comum de iniciantes

Começar direto em uma linguagem sem entender lógica.

Isso gera:
- Confusão
- Dificuldade de aprender
- Dependência de copiar código

➡️ Primeiro lógica, depois linguagem.

---

## 🛠️ Prática

A melhor forma de aprender lógica é praticando.

### Sugestões:
- Resolver problemas simples
- Criar algoritmos do dia a dia
- Treinar raciocínio

---

## 🚀 Próximo passo

Depois de dominar lógica, você estará pronto para:

- Aprender qualquer linguagem
- Resolver problemas reais
- Evoluir rapidamente na programação

---

## 🎯 Conclusão

Lógica de programação é a base de tudo.

Se você dominar isso:
- Linguagens se tornam ferramentas
- Problemas se tornam desafios resolvíveis

➡️ Quem domina lógica, aprende qualquer tecnologia.
