# 🗣️ O que é uma Linguagem de Programação?

## 💡 Introdução

Ao iniciar na programação, é comum surgir a dúvida:

➡️ **O que é uma linguagem de programação?**

Computadores não entendem português, inglês ou qualquer idioma humano.
Eles operam em um nível muito mais básico, utilizando **código binário (0 e 1)**.

Para que possamos nos comunicar com essas máquinas, utilizamos um intermediário:

➡️ **A linguagem de programação**

---

## 🧠 Definição

Uma linguagem de programação é um conjunto de:

- Palavras
- Símbolos
- Regras

que permitem escrever instruções que o computador consegue interpretar e executar.

➡️ É como aprender um novo idioma — só que para conversar com máquinas.

---

## 🎨 Programação também pode ser criativa

Existem linguagens curiosas e até inusitadas.

### Exemplo:
- **Emojicode** 😄🔥💻

➡️ Uma linguagem feita com emojis.

Apesar de não ser usada no mercado, ela mostra que a programação também pode ser:
- Criativa
- Divertida
- Fora do padrão

---

## ⚠️ Errar faz parte do processo

Durante sua jornada, você vai encontrar:

- Bugs 🐛
- Erros no código
- Mensagens difíceis de entender

➡️ Isso é normal.

Programar envolve:
- Testar
- Errar
- Corrigir
- Aprender

---

## 🔍 Saber pesquisar é essencial

Na programação, você não precisa saber tudo.

O mais importante é saber:
- Buscar soluções
- Ler documentação
- Testar alternativas

➡️ Programar é resolver problemas.

---
